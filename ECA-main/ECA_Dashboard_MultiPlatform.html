Autodesk Inventor — iLogic Rule (ECA_Inventor.ilogic)
VB.NET · COM API
  • 1
    Open ValveBody.ipt in Autodesk Inventor 2020+
    Ensure user parameter "WallThickness = 12mm" exists in Part Parameters
  • 2
    Open iLogic Browser: Tools → iLogic → iLogic Browser
    Right-click "Rules" → Add Rule → name it "ECA_Phase2"
  • 3
    Paste entire ECA_Inventor.ilogic content into the rule editor
  • 4
    Click "Run Rule" button (▶) in iLogic Browser
    Or set trigger: Part Events → OnSave to auto-run on every save

Fusion 360 — Python Add-In (ECA_Fusion360.py)

    Files exported to Desktop: .step · .f3d · .iges
